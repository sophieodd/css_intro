### рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ: "Introduction to CSS" - рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ

**рЃљрЃдрЃгрЃћрЃарЃљ:**
рЃљрЃљрЃгрЃДрЃЋрЃћрЃЌ рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџрЃў рЃЊрЃўрЃќрЃљрЃўрЃюрЃў HTML/CSS-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ. рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ рЃАрЃБрЃцрЃЌрЃљ рЃЊрЃљ рЃЋрЃљрЃџрЃўрЃЊрЃБрЃарЃў рЃЎрЃЮрЃЊрЃўрЃА рЃгрЃћрЃарЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃљрЃЊ рЃќрЃБрЃАрЃбрЃљрЃЊ рЃўрЃЏрЃћрЃЮрЃарЃћрЃЉрЃА рЃЊрЃўрЃќрЃљрЃўрЃюрЃА.

#### ­ЪЏа рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃў (Technical Requirements)

1. **Semantic HTML:** рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃАрЃћрЃЏрЃљрЃюрЃбрЃўрЃЎрЃБрЃарЃў рЃбрЃћрЃњрЃћрЃЉрЃў (`<header>`, `<main>`, `<section>`, `<footer>`, `<button>`) `<div>`-рЃћрЃЉрЃўрЃА рЃќрЃћрЃЊрЃЏрЃћрЃбрЃў рЃЊрЃљ рЃљрЃарЃљрЃАрЃљрЃГрЃўрЃарЃЮ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃюрЃљрЃфрЃЋрЃџрЃљрЃЊ.
2. **Selector Specificity:** рЃАрЃбрЃўрЃџрЃўрЃќрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў (`class`). рЃЏрЃЮрЃћрЃарЃўрЃЊрЃћрЃЌ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃбрЃћрЃњрЃўрЃЌ (рЃЏрЃљрЃњ: `div p`) рЃљрЃю ID-рЃўрЃЌ рЃАрЃбрЃўрЃџрЃўрЃќрЃћрЃЉрЃљрЃА, рЃарЃљрЃЌрЃљ рЃЎрЃЮрЃЊрЃў рЃўрЃДрЃЮрЃА рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљрЃЊрЃў.
3. **Unique IDs:** **рЃЏрЃЎрЃљрЃфрЃарЃљрЃЊ рЃЊрЃљрЃўрЃфрЃљрЃЋрЃўрЃЌ ID-рЃћрЃЉрЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃЮрЃЉрЃљ.** рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ рЃљрЃа рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃЮрЃарЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃћрЃарЃЌрЃў рЃЊрЃљ рЃўрЃњрЃўрЃЋрЃћ `id` рЃЊрЃљрЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃўрЃЌ.
4. **Naming:** рЃЎрЃџрЃљрЃАрЃћрЃЉрЃАрЃљ рЃЊрЃљ ID-рЃћрЃЉрЃА рЃЊрЃљрЃљрЃарЃЦрЃЋрЃўрЃЌ рЃерЃўрЃюрЃљрЃљрЃарЃАрЃЮрЃЉрЃарЃўрЃЋрЃў (Descriptive) рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃў рЃўрЃюрЃњрЃџрЃўрЃАрЃБрЃа рЃћрЃюрЃљрЃќрЃћ.

#### ­ЪЊё рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ (Readme Requirement)

**рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮрЃљ**, рЃарЃЮрЃЏ `README.md` рЃцрЃљрЃўрЃџрЃерЃў рЃЕрЃљрЃюрЃЊрЃћрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃюрЃљрЃЏрЃБрЃерЃћрЃЋрЃарЃўрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃў (рЃАрЃЦрЃарЃўрЃюрЃерЃЮрЃЌрЃў).

**рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ:**

1. рЃњрЃљрЃЊрЃљрЃўрЃдрЃћрЃЌ рЃљрЃгрЃДрЃЮрЃЉрЃўрЃџрЃў рЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃАрЃБрЃарЃљрЃЌрЃў (Screenshot) рЃЊрЃљ рЃЊрЃљрЃљрЃарЃЦрЃЋрЃўрЃЌ рЃАрЃљрЃ«рЃћрЃџрЃў (`preview.png`).
2. рЃљрЃбрЃЋрЃўрЃарЃЌрЃћрЃЌ рЃћрЃА рЃАрЃБрЃарЃљрЃЌрЃў рЃЌрЃЦрЃЋрЃћрЃюрЃА GitHub рЃарЃћрЃърЃЮрЃќрЃўрЃбрЃЮрЃарЃўрЃљрЃерЃў (рЃЎрЃЮрЃЊрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ).
